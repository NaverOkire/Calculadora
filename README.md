# 💻 Calculadora & Gerenciador Financeiro

Aplicação web **full-stack** desenvolvida com foco na evolução prática da lógica de programação, construção de sistemas completos e integração entre frontend, backend e banco de dados.

O projeto iniciou como uma calculadora simples e evoluiu para um sistema com **autenticação, persistência de dados e gerenciamento de contas**.

---

## 🚀 Funcionalidades

### 🔢 Calculadora

- Operações básicas: `+`, `-`, `×`, `÷`
- Suporte a parênteses e múltiplas operações
- Tratamento de porcentagem (`%`)
- Engine baseada em expressões
- Histórico interativo de cálculos
- Suporte completo ao teclado
- Backspace
- Dispatch central de operadores

### 🧾 Gerenciador de Contas (Bills Manager)

Sistema completo de gerenciamento de contas integrado ao usuário:

- CRUD completo:
  - Criar
  - Listar
  - Atualizar
  - Deletar
- Associação com usuário autenticado
- Persistência em banco de dados (MySQL)
- Validação de dados no backend
- Tratamento de erros estruturado
- Feedback de erro no frontend
- Resumos e organização das contas
- Interface dedicada no painel principal

### 🔐 Autenticação

- Registro de usuários
- Login com autenticação via JWT
- Senhas protegidas com bcrypt
- Rotas protegidas via middleware
- Sessão integrada com frontend
