💻 Calculadora & Gerenciador Financeiro
Aplicação web full-stack desenvolvida com foco em evolução prática de lógica de programação, construção de sistemas completos e integração entre frontend, backend e banco de dados.
O projeto iniciou como uma calculadora simples e evoluiu para um gerenciador de contas completo com autenticação de usuários e persistência de dados.

🚀 Funcionalidades
🔢 Calculadora
Operações básicas: +, -, ×, ÷

Suporte a parênteses e múltiplas operações

Tratamento de porcentagem (%)

Histórico interativo de cálculos

Suporte ao teclado e Backspace

Ajuste automático de fonte e "clique para copiar"

📝 Gerenciador de Contas (Bills To-Do)
CRUD completo: Criar, listar, atualizar e deletar contas/pendências

Painel de gerenciamento integrado à interface principal

Dados vinculados exclusivamente ao usuário autenticado

Validação de inputs e tratamento de erros no frontend e backend

Resumos e feedbacks visuais de operações

🔐 Autenticação & Segurança
Registro e Login de usuários

Autenticação via JWT com HttpOnly Cookies

Senhas protegidas com hash (bcrypt)

Rotas protegidas por middleware no backend

Proteção contra força bruta via Rate Limiting

🎨 Interface & Experiência
Tema claro/escuro com persistência

Layout responsivo e otimizado para mobile

Nova paleta de cores com botões em gradiente

Animações de interface e transições de módulos

🛠️ Tecnologias
Frontend

HTML5, CSS3 e JavaScript (ES6 Modules)

Backend

Node.js e Express

Banco de Dados

MySQL (Schema relacional com bills_todo)

Segurança

JWT, bcrypt e Rate Limit

⚙️ Arquitetura
Backend em Node.js com API REST

Separação em controllers, routes e middlewares

Frontend modularizado (BillsApp) consumindo API via requisições HTTP

Persistência de sessão e segurança de cookies aprimorada

🎯 Objetivo
Evoluir de um projeto simples para uma aplicação full-stack funcional

Consolidar fundamentos de banco de dados relacional e segurança

Desenvolver uma interface intuitiva para gestão de tarefas financeiras
